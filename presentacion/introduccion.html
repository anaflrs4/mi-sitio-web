Introducción
Objetivo del Proyecto
Desarrollar un sitio web por medio de los elementos del diseño web para el posicionamiento de una marca personal.
Este proyecto permite aplicar conocimientos de HTML, CSS y JavaScript para crear una presencia web profesional y atractiva.
Competencias a Desarrollar
Sello EBC: Emprendimiento-innovación
Técnica: Uso de herramientas de diseño web
Laboral: Organización
