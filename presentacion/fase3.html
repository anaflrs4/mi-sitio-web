Fase 3: Implementación y estilo del sitio web
Elementos CSS3 implementados
Media queries - Para adaptación a diferentes dispositivos
Flexbox - Para la disposición de elementos
Transiciones - Para efectos visuales
Variables CSS - Para mantener consistencia en colores y tipografía
Box-sizing - Para un manejo más intuitivo de las dimensiones
Ejemplo de código CSS
/* Media queries para responsiveness */
@media (max-width: 768px) {
  header h1 {
    font-size: 24px;
  }
}

@media (max-width: 480px) {
  header h1 {
    font-size: 20px;
  }
  main {
    padding: 10px;
  }
}
Diseño responsivo
El diseño implementado se adapta a diferentes dispositivos, garantizando una experiencia de usuario óptima tanto en equipos de escritorio como en dispositivos móviles.
Vista del sitio web
Características del diseño:
  • Estructura clara y organizada
  • Tipografía legible en todos los dispositivos
  • Espaciado adecuado entre elementos
  • Contraste de colores para mejor legibilidad
