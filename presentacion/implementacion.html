Implementación del Sitio Web

Fase 1: Configuración del Repositorio
Creación del repositorio en GitHub Pages y configuración inicial del proyecto.
Fase 2: Estructura HTML
Desarrollo de la estructura HTML5 para cada página siguiendo los wireframes proporcionados.
Fase 3: Diseño CSS
Implementación de estilos CSS3 con media queries, flexbox, transiciones, variables CSS y box-sizing.
Fase 4: Funcionalidades JavaScript
Integración de la biblioteca AOS para animaciones y desarrollo de elementos interactivos.
Fase 5: Framework Bootstrap
Integración de Bootstrap 5.1 para mejorar el diseño y la responsividad del sitio.
Fase 6: Contenido y Navegación
Implementación del contenido detallado para cada sección y configuración de la navegación entre páginas.
